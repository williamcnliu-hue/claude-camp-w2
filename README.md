## 🎯 练习清单

| 文件 | 功能 | 核心概念 |
|------|------|---------|
| `student_roster.py` | 学员花名册管理器 | 字典、列表、while 循环、if/elif、布尔标志 |
| `word_counter.py` | 文本词频统计器 | 字符串处理、字典统计、排序、lambda |
| `todo_list.py` | 待办事项清单（带文件保存） | 函数定义、JSON 文件读写、try/except FileNotFoundError |
